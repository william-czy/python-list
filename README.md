# Python List

## List函数

- append
- insert
- remove
- pop
- sort
- reverse
- clear

## 代码输出
```
> python3 list.py
there are 3 fruits, they are: ['banana', 'apple', 'cherry']
list.append  - there are 4 fruits, they are: ['banana', 'apple', 'cherry', 'orange']
list.insert  - there are 5 fruits, they are: ['banana', 'watermelon', 'apple', 'cherry', 'orange']
list.remove  - there are 4 fruits, they are: ['banana', 'watermelon', 'apple', 'orange']
list.pop(1)  - there are 3 fruits, they are: ['banana', 'apple', 'orange']
list.sort    - there are 3 fruits, they are: ['apple', 'banana', 'orange']
list.reverse - there are 3 fruits, they are: ['orange', 'banana', 'apple']
list.clear   - there are 0 fruits, they are: []
```
